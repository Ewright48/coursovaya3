# Flower Shop — Интернет-магазин для продажи букетов

## Описание проекта

Интернет-магазин цветов с возможностью просмотра каталога, фильтрации, добавления товаров в корзину, оформления заказов и управления личным кабинетом.

**Основные возможности:**
- Просмотр каталога с фильтрацией и сортировкой
- Добавление товаров в корзину
- Оформление заказа с выбором доставки
- Авторизация и регистрация
- Личный кабинет с историей заказов

## Архитектура проекта
Клиент (Vue SPA) ←→ REST API (Express) ←→ База данных (PostgreSQL)

**Компоненты:**
- **Фронтенд:** Vue 3, Vue Router, Tailwind CSS 4.2
- **Бэкенд:** Node.js, Express, JWT, bcrypt, dotenv
- **БД:** PostgreSQL

## Инструкция по развертыванию и запуску

### Требования
- Node.js (v16+)
- PostgreSQL (v18+)
- Git

### 1. Клонирование репозитория
```bash
git clone https://github.com/username/flower-shop.git
cd flower-shop
```
### 2. Настройка базы данных
``` bash
psql -U postgres -f BD_flowers_fill.sql
```
### 3. Настройка переменных окружения
``` bash
cp backend/.env.example backend/.env
```
Отредактируйте backend/.env (укажите пароль от БД)
### 4. Установка зависимостей и запуск
``` bash
npm run install:all
npm run dev
```
### 5. Доступ к приложению
Фронтенд: http://localhost:5173

API: http://localhost:5000/api
